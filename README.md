# D3课程
- D3 v3.5版
- 基于underscore.js 函数是编程
- webpack4.0(解决丑化 调试时自动更新 ES6语法支持 低版本IE支持等一系列问题)

## Start
 - NodeJS 8.0 need
 - Clone or download this repository
 - Enter your local directory, and install dependencies:

``` bash
npm install
npm start
```
如果不能自动打开浏览器 请浏览器访问: http://localhost:4000
